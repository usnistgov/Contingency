# Installation

`contingency` is currently under review for release on the PyPi repository.
Until then, a preview may be installed directly from github:

```shell
pip install git+https://github.com/usnistgov/Contingency.git
```


## Optional Plotting Utilities
Install the extra [plotting utilities](./02-tutorial.md#optional-plotting-utilities) (Matplotlib required).

```shell
pip install "git+https://github.com/usnistgov/Contingency.git[plot]"
```
